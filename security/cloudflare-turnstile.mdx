    receive through your contact form, it will not stop 100% of it, however, it will stop
    the vast majority of automatically generated spam.
</Info>

## What is Cloudflare Turnstile

## How to enable Cloudflare Turnstile

Cloudflare Turnstile will automatically enable if you provide the necessary API keys.

### Getting the Cloudflare Turnstile API keys

Step 1. Go to your Cloudflare Dashboard
Step 2. Go to the Turnstile section
Step 3. Create a new site
Step 4. Specify your domain 

### Adding your API keys to Site Seedling

Step 1. Go to the Edit your website section
Step 2. Go to `Integrations`
Step 3. Paste in the Turnstile site key and secret key into the relevant fields
Step 4. Save

Once you have done that your site will autoamtically start being protected by Cloudflare Turnstile
